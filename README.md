# JINGYUE-X99-D4-OpenCore

## 配置

主板：`JINGYUE X99M-PLUS D4`

处理器：`Intel(R) Xeon(R) CPU E5-2683 v4 @ 2.10GHz`

图形卡：`AMD Radeon RX 560 4 GB` 盈通`rx560d`

内存：`64 GB 2133 MHz DDR4`

`MacOS`：`13.2 (22D49)`

网卡和蓝牙：`Intel AC-9260`

`OpenCore`：`0.8.8`
